# C-FIND : Cardio-fibrosis Innovative Novel Drug Development
## Dependencies
* Docker >= 27.3.1
## How to use
```bash
$ git clone https://github.com/MOJAN3543/cfind.git

# Go into the repository
$ cd cfind

# Build Docker Image
$ docker-compose up -d --build
```
## Stop & Remove Container
```bash
# Stop and Remove Container
$ docker-compose down
```
